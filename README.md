# Skladový systém (C#)

Aplikace pro správu skladových zásob v C# - Windows Forms a SQLite databáze.

---

## Funkce
- Přidání produktu  
- Úprava produktu  
- Smazání produktu  
- Vyhledávání podle názvu nebo kategorie  
- Přehled všech produktů  

---

## Použité technologie
- C# (.NET)
- Windows Forms
- SQLite
- Microsoft.Data.Sqlite (NuGet)
